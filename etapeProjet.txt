Project: s'inflitrer dans les server d'une entreprise.

Description: À partir d'un site web, trouver un moyen de s'inflitrer dans le server


étape:
- essayer de voir un VPN ou comment on peut modifier son adresse IP
- avoir l'IP du site, via le DNS
- essayer faire un scan des ports du site web pour trouver des ports ouverts 
- essayer d'envoyer plusieurs requêtes avec un nom de page différent (utiliser un dictionnaire) 
  pour voir si il n'y a pas des informations intéressantes 
- on va trier les informations grâce à des mots clés (version, configuration, etc...)
(étapes facultatives)
- demander les logs de la personne pour chatgpt
- avec les informations trouvées essayer de faire appelle à à une libraire d'intelligence artificielle (chatgpt)
  trouver un moyen de se connecter à chatgpt en entrant les logs.
- faire une demande à chatgpt selon les informations reçu (demande de recherche de vulnérabilité de version ou de configuration)
- afficher les promptes de chatgpt à l'écran

(si les étapes facultatives ne sont pas possible)
- faire les étapes facultatives à la main
